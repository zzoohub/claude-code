# Ad Creative

Ad creative development, copy generation, and batch production across platforms.

---

## Ad Creative Framework

### 1. Define the Brief
- **Audience**: Who are we targeting? What do they care about?
- **Objective**: Awareness, consideration, or conversion?
- **Platform**: Where will this run? (each has different specs and norms)
- **Message**: What is the ONE thing we want to communicate?
- **CTA**: What action do we want them to take?

### 2. Create the Ad Structure
Every ad follows: **Hook → Value Prop → CTA**
- **Hook** (first 3 seconds / first line): Grab attention
- **Value Prop**: Why should they care? What do they get?
- **CTA**: Clear, single action

### 3. Generate Variations
For every ad, create:
- 3-5 hook variations (different angles, different emotions)
- 2-3 value prop angles (benefit, proof, story)
- 2 CTA variations (direct vs soft)
- Combine for a test matrix of 10-15 variations

### 4. Iterate Based on Performance
- Kill underperformers after sufficient data
- Scale winners
- Create new variations based on winning elements

## Ad Types Overview

### Search Ads (Google/Bing)
- **Format**: Headlines (30 chars each) + Descriptions (90 chars each)
- **Key**: Match user search intent exactly
- **Tip**: Include the search keyword in headline 1

### Social Ads (Facebook/Instagram/LinkedIn)
- **Format**: Image/video + Primary text + Headline + Description
- **Key**: Stop the scroll with visual + hook
- **Tip**: First line of primary text is everything

### Display Ads
- **Format**: Banner images in standard sizes (300x250, 728x90, 160x600)
- **Key**: Simple message, clear CTA, brand visible
- **Tip**: Less is more — one message, one image, one CTA

### Video Ads
- **Format**: 6s bumper, 15s, 30s, 60s
- **Key**: Hook in first 3 seconds, value by 5s, CTA at end
- **Tip**: Design for sound-off (captions/text overlay)

## Core Ad Copy Principles

- **Hook in first 3 seconds / first line** — You have zero margin
- **One message per ad** — Don't try to say everything
- **Specific benefits over generic claims** — "Save 4 hours/week" not "Save time"
- **Clear, single CTA** — One action, one button
- **Message match** — Ad message must match landing page headline
- **Test variations systematically** — Don't guess; test hooks, angles, CTAs

---

## Variation Generation Process

1. **Identify the winning angle** — Problem, benefit, social proof, curiosity, urgency
2. **Write 3-5 hook variations** per angle
3. **Write 2-3 body variations** — Different value props
4. **Write 2 CTA variations** — Direct ("Start free trial") vs soft ("See how it works")
5. **Combine** — Create test matrix
6. **Review** — Check message match with landing page
